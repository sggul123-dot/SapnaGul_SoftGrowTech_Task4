# SapnaGul_SoftGrowTech_Task3
Basic Chatbot – Task 4
Description

This project is a simple rule-based chatbot developed in Python.
The chatbot responds to predefined user inputs using conditional statements.

Features

Takes user input from console

Responds to specific messages

Runs continuously until the user types "bye"

Simple text-based interaction

Supported Inputs

hello

how are you

bye

Responses

hello → Hi!

how are you → I'm fine, thanks!

bye → Goodbye!

Other inputs → I don't understand that.

Concepts Used

if-elif statements

Functions

while loop

Input/output

Object-Oriented Programming (Class)

How to Run

Open terminal or command prompt.

Navigate to the project folder.

Run the command:

python BasicChatbot.py

Type messages like "hello", "how are you", or "bye".

BasicChatbot.py
